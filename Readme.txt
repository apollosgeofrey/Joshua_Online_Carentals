How to run Project
1. Download and Unzip the file on your local system copy joshua_online_carentals .
2. Put joshua_online_carentals folder inside root directory (for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

Database Configuration

Open phpmyadmin
Create Database joshua_online_carentals
Import database joshua_online_carentals.sql (available SQL File Folder inside zip package)
